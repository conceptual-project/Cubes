# Кубики / Cubes

**Кубики** — это игра, в которой нужно удалять группы одинаковых кубиков с игрового поля. Чем больше группа — тем больше очков! Игра работает полностью офлайн как PWA.

## 🎮 Об игре

Поле 20×15 заполнено кубиками четырёх цветов. Кликните на группу из двух и более соседних кубиков одного цвета, чтобы удалить их. Кубики сверху падают вниз, а пустые столбцы сдвигаются влево. Цель — набрать как можно больше очков и очистить поле!

### Ключевые механики

- **Групповое удаление** — кликайте по группе одинаковых кубиков, чтобы удалить её
- **Бонус за размер** — чем больше группа, тем выше множитель очков (n × (n − 1), где n — размер группы)
- **Бонус за очистку столбца** — +150 очков за каждый полностью очищенный столбец
- **Идеальная очистка** — +2000 очков, если удалены абсолютно все кубики!
- **Отмена хода** — можно отменить до 20 последних ходов
- **Таблица рекордов** — топ-10 лучших результатов сохраняется локально
- **Повтор** — можно переиграть партию с тем же расположением кубиков
- **Звуковые эффекты** — синтезированные звуки через Web Audio API (можно отключить)
- **Двуязычность** — русский / английский интерфейс

## 🕹 Как играть

1. Нажмите **ИГРАТЬ**, чтобы начать новую игру
2. Наведите курсор на кубик — подсветится вся группа одинаковых соседних кубиков
3. Кликните, чтобы удалить группу
4. Набирайте очки: за группу из n кубиков даётся n × (n − 1) очков
5. Старайтесь создавать как можно более крупные группы для максимума очков
6. Используйте кнопку **↩** для отмены последнего хода
7. Игра заканчивается, когда на поле не осталось групп из 2+ кубиков

## 📱 Установка как PWA

### На мобильном устройстве

1. Откройте игру в браузере (Chrome, Safari, Firefox)
2. **Android (Chrome):** нажмите ⋮ → «Установить приложение»
3. **iOS (Safari):** нажмите  → «На экран "Домой"»
4. Игра появится на рабочем столе и будет работать офлайн

### На десктопе

1. Откройте игру в Chrome или Edge
2. Нажмите иконку установки в адресной строке или ⋮ → «Установить Кубики»
3. Игра установится как отдельное приложение
