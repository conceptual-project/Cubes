# Кубики

Легковесная игра «Кубики». Работает оффлайн.

## ✨ Особенности

- **Топ-10 игроков:** локальная таблица лидеров с сохранением результатов
- **Управление состоянием:** счётчик очков, выделенные элементы, лимит ходов (300)
- **Мультиязычность:** переключение интерфейса EN/RU
- **Мгновенный рестарт:** кнопки «Заново» для быстрого сброса партии
- **Управление:** правая кнопка мыши отменяет ход
