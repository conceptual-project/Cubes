# 🎲 Кубики (Cubes PWA)

Легковесная браузерная игра «Кубики». Работает оффлайн, устанавливается на рабочий стол компьютера или домашний экран смартфона без использования магазинов приложений.

## ✨ Особенности

- **Топ-10 игроков:** локальная таблица лидеров с сохранением результатов
- **Управление состоянием:** счётчик очков, выделенные элементы, лимит ходов (300)
- **Мультиязычность:** переключение интерфейса EN/RU
- **Мгновенный рестарт:** кнопки «Заново» и «Отмена» для быстрого сброса партии
